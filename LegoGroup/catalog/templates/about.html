About


BrickViewer is an application which aims to provide access to a list of Lego parts for users to browse. Pieces are organized based on the category and sub-category they belong to. For example, users can view regular or modern electric bricks.

The application is secure against XSS since the Django template system is used for the HTML part (as specified here ).
